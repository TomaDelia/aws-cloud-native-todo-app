# Project Overview

This project is a cloud-native ToDo application developed as part of my ITS Academy ICT final thesis and internship experience.

The goal of the project is to design and implement a scalable backend system using AWS cloud services, containerization, and CI/CD automation.

## Main Focus Areas

- Cloud architecture design on AWS
- Backend development using .NET
- Containerization with Docker
- Deployment automation using CI/CD pipelines
- Integration of managed AWS services

## Architecture Summary

The system is composed of:

- Frontend hosted on S3 and distributed via CloudFront
- Backend deployed on ECS Fargate containers
- Database hosted on Amazon RDS
- API Gateway as entry point
- CI/CD pipeline using CodePipeline and CodeBuild

## Purpose

The project was designed to simulate a production-ready cloud architecture and to gain hands-on experience with AWS services used in real-world environments.
